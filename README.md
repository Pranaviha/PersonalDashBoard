A clean, all-in-one personal productivity dashboard designed to help you take control of your day — from the moment you wake up to the moment you wind down.
Key features include:

* Live clock & date display
* Real-time local weather
* Pomodoro timer with work/break modes
* To-do list with task management
* Notes panel for quick thoughts
* Habit tracker with daily check-ins
* Productivity stats overview
* Monthly calendar view
